# SimpleSwitchList

## Сборка

### Линт проекта:
   ```bash
   # lint
   npm run lint
   ```

### Среда разработки (без сборки нативного кода):
   ```bash
   # iOS
   npx expo start --ios
   ```
   ```bash
   # Android
   npx expo start --android
   ```
   ```bash
   # Web
   npm run web
   ```

### Локальная сборка:
   ```bash
   # iOS (debug)
   npm run build:ios:develop
   ```
   ```bash
   # Android (debug)
   npm run build:android:develop
   ```
   ```bash
   # iOS (release)
   npm run build:ios:release
   ```
   ```bash
   # Android (release)
   npm run build:android:release
   ```
   ```bash
   # Web (release)
   npm run build:web
   ```
